TASKFLOW!

TaskFlow is a fully functional task management web application that helps users organize their personal and work-related tasks efficiently. It allows individuals to securely sign up and manage their own set of tasks, with the ability to add, edit, delete, and mark them as completed. Users can filter tasks by status, priority, or category, and see real-time summaries like total, completed, active, and high-priority tasks. The application is optimized for both desktop and mobile, offering a smooth experience with a slide-out drawer on smaller screens for quick access to filters and stats. The design emphasizes simplicity, responsiveness, and ease of use while ensuring all data remains private and tied to the authenticated user.
Tech Stack: React, Firebase (Auth + Firestore), Tailwind CSS, Lucide Icons
